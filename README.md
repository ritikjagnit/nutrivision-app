# 🍏 NutriVision – Smart Food Analyzer

NutriVision is an AI-powered web application that detects food items from images and provides basic nutritional insights. The goal of this project is to help users understand what they are eating and make smarter dietary decisions.

---

## 🚀 Features

* 📸 Upload food images
* 🤖 AI-based food detection
* 🥪 Displays food name with basic information
* ⚡ Fast and responsive UI
* 🌐 Frontend built with React
* 🔙 Backend powered by Python

---

## 🛠️ Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Python

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/ritikjagnit/nutrivision.git
cd nutrivision
```

---

### 2️⃣ Setup Backend (Python)

```
cd backend
pip install -r requirements.txt
python app.py
```

---

### 3️⃣ Setup Frontend (React)

```
cd frontend
npm install
npm start
```

---

## 📸 How It Works

1. User uploads a food image
2. Image is sent to backend API
3. Model processes the image
4. Food name + details are returned
5. Results displayed on UI

---
